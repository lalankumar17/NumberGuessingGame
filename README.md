# Number Guessing Game 

## Intro

> The computer generates a random number between **1 and 50**. Your goal is to guess the number in the minimum number of chances!

---


1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Build the project**
   ```bash
   npm run build
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
